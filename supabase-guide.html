Supabase Integration Hub

Connect your cloud database, install the schema, and migrate simulated data in real time

01
🌐

Create a Free Supabase Project

If you haven't yet, head over to supabase.com, log in with your GitHub account, and click New Project. Choose a project name, secure database password, and launch your dedicated cloud PostgreSQL instance (takes 1-2 minutes to provision).

02
💾

Install the Database Schema

Open your project inside the Supabase Console, navigate to the SQL Editor on the left-hand menu, click New Query, paste the full database structure below, and click Run. This sets up your tables, auto-increment ticket sequences, and automatic row triggers!

03
🔑

Connect Your Web Application

Inside your Supabase dashboard, go to Project Settings (the gear icon) > API. Copy your Project URL and your anon public key.

Configure Credentials for This Browser Session

💡 Pro Tip: To permanently configure these credentials for all users without needing local browser settings, open js/supabase-client.js and update LIVE_SUPABASE_URL and LIVE_SUPABASE_ANON_KEY at the top of the file!
